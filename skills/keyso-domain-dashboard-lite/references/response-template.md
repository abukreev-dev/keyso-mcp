# Domain Snapshot Template

Use for very short responses.

- Scope: `<domain>` in `<base>`
- DR: `<dr>`
- Visibility: `<vis>`
- Keywords: top10 `<it10>`, top50 `<it50>`
- Paid estimate: traffic `<adtraf>`, budget `<adcost>`
- AI mentions: `<aiAnswersCnt>` (state: `<aiState>`)
- Next step (optional): one single suggestion
