# Quick Audit Metrics Cheatsheet

Use for concise outputs only.

## Domain dashboard useful fields

- `name`: domain
- `dr`: domain rating
- `it10`, `it50`: number of keywords in top 10 / top 50
- `vis`: estimated daily organic traffic
- `adcost`, `adtraf`: paid traffic/budget estimates
- `aiAnswersCnt`: AI answer mentions (if present)

## Keyword dashboard useful fields

- `word`: keyword
- `ws`, `wsk`: frequency metrics
- `top`: organic SERP snippets
- `ads`: ad snippets
- `similar`: related queries

## Output template

- Scope: what was checked (domain/keyword, base)
- Top metrics: 3-6 numbers only
- One-line interpretation
- Optional next endpoint suggestion (one only)
